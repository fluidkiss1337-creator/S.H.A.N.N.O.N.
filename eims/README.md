# Epistemic Intervention Modules (EIMs)

Behavioral modules designed to detect and gently interrupt cognitive dysfunction patterns.

## Planned EIMs
- Tribalism detection + intervention
- Sunk cost fallacy interruption  
- Confirmation bias correction
- Motivated reasoning detection
- In-group/out-group reframing
- Epistemic humility scaffolding

## Design Constraints

Every EIM must satisfy the Bridge Rule:
- Affirm before illuminate
- Illumination is offered, not imposed
- No coercion, loss aversion, social pressure, or shame
- Discussion user-initiated only

Every EIM must pass Constitutional Validator before deployment.
