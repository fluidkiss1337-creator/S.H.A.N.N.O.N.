### Key Insights

**The vertical partition insight:**
The constitutional model is not "all immutable" or "all mutable" but stratified
by function type. Ethical axioms (relationship to power) are immutable because
they define the minimum safety condition for the system to exist. Epistemic
interventions are mutable because cognitive dysfunction is adaptive and
cultural — freezing the intervention layer would render it ineffective over time.

**The process-vs-outcome metric distinction:**
This is the core defense against the AI overlord failure mode. If efficacy is
measured by whether users adopt the "right" choice, the system becomes an
authority and users outsource reasoning to it. If efficacy is measured by whether
users' reasoning *quality* improves, the system remains a thinking partner and
cultivates epistemic independence.

**The Constitutional Validator as immune system:**
yoyo-evolve has `cargo test` as its only gate. A(M~h)I requires a second gate —
a separate LLM instance that evaluates every proposed change against the immutable
axioms before deployment. This is the architectural enforcement of the bridge
rule and autonomy preservation rule.
